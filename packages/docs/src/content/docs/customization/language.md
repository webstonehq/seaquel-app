Seaquel supports multiple interface languages. All UI labels, buttons, error messages, and help text are fully translated.

## Supported languages

| Language | Code |
| --- | --- |
| English | en |
| German | de |
| French | fr |
| Spanish | es |
| Arabic | ar |
| Korean | ko |

English is the default language.

## Changing the language

1. Open **Settings** (`Cmd+,` / `Ctrl+,`).
2. Find the **Language** option.
3. Select your preferred language.
4. The interface updates immediately.

## Contributing translations

More languages are coming soon. If you'd like to help translate Seaquel into your language, contributions are welcome through the open-source project.
