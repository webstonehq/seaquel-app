Every action in Seaquel has a keyboard shortcut. Mac shows `Cmd`, Windows and Linux show `Ctrl`. Press `?` from anywhere in the app to view the full shortcut list.

## General

<Tabs>
<TabItem label="Mac">

| Action | Shortcut |
| --- | --- |
| Open command palette | `Cmd+K` |
| Show keyboard shortcuts | `?` |
| Open settings | `Cmd+,` |

</TabItem>
<TabItem label="Windows / Linux">

| Action | Shortcut |
| --- | --- |
| Open command palette | `Ctrl+K` |
| Show keyboard shortcuts | `?` |
| Open settings | `Ctrl+,` |

</TabItem>
</Tabs>

## Tab management

<Tabs>
<TabItem label="Mac">

| Action | Shortcut |
| --- | --- |
| New query tab | `Cmd+T` |
| Close current tab | `Cmd+W` |
| Next tab | `Cmd+Shift+]` |
| Previous tab | `Cmd+Shift+[` |
| Go to tab 1-9 | `Cmd+1` through `Cmd+9` |

</TabItem>
<TabItem label="Windows / Linux">

| Action | Shortcut |
| --- | --- |
| New query tab | `Ctrl+T` |
| Close current tab | `Ctrl+W` |
| Next tab | `Ctrl+Shift+]` |
| Previous tab | `Ctrl+Shift+[` |
| Go to tab 1-9 | `Ctrl+1` through `Ctrl+9` |

</TabItem>
</Tabs>
