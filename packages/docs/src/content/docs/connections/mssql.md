title: Microsoft SQL Server
description: Connect to SQL Server and Azure SQL with Seaquel.
---

Seaquel supports Microsoft SQL Server connections in the desktop app.

## Connection details

| Field | Default |
| --- | --- |
| Host | `localhost` |
| Port | `1433` |
| Database | — |

## Authentication

SQL Server authentication with username and password. Credentials are stored securely in your system keychain.

## SSL / TLS

Enable SSL/TLS encryption for connections over the network. Recommended for production and cloud-hosted databases.

## Azure SQL Database

Seaquel works with Azure SQL Database. Use the connection details from the Azure portal — server name, database name, and SQL authentication credentials.

## Desktop only

SQL Server connections are available in the desktop app. The demo and web versions do not include MSSQL support.
